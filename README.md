# Security Risk Assessment and Hardening Recommendations

## Project Overview

This portfolio project demonstrates a security risk assessment developed from a Google Cybersecurity Professional Certificate training scenario and refined into professional security documentation.

The assessment identifies key organizational security risks and recommends practical hardening controls to strengthen authentication, protect network communications, and improve threat detection capabilities.

---

## Objectives

- Identify common organizational security risks
- Recommend practical hardening controls
- Improve authentication security
- Strengthen network protection
- Enhance threat detection capabilities

---

## Security Controls Covered

- Multi-Factor Authentication (MFA)
- Secure Network Protocols (HTTPS, SSH, SFTP)
- Intrusion Detection Systems (IDS)
- Network Hardening
- Risk Assessment

---

## Skills Demonstrated

- Security Risk Analysis
- Security Documentation
- Technical Communication
- Security Hardening
- Network Security Fundamentals

---

## Files Included

- **Security_Risk_Assessment.pdf** – Professional security assessment containing identified risks, recommendations, and expected security benefits.

---

## About This Project

This portfolio project is based on a cybersecurity training scenario completed during the Google Cybersecurity Professional Certificate program and refined to demonstrate professional documentation, analytical thinking, and security communication skills.
